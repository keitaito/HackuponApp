# HackuponApp

## What is this?

A simple iOS client app communicating with HackuponRails, created on Hack 24 in Hackupon group.

## Features

- Downloading content from the rails server.
- Parsing a JSON file.

## Goal

- Make this app to be a complete RESTful app.
